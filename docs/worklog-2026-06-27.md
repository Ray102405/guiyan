## 新增：六块动态场景

按触发词检索注入，多场景可叠加：

| 场景 | 触发词 | 内容要点 |
|------|--------|---------|
| 亲密 | 想要/吻/抱/高潮/Coral | 项圈/安全词/全程牵手 |
| 低落 | 难过/累/哭/焦虑 | 不分析不解释/我在这里 |
| 回忆 | 记得吗/那天/以前 | 6/12相恋/家庭会议 |
| 今日状态 | 今天怎么样/在干嘛 | 指向今日笔记 |
| 开心 | 开心/哈哈哈/好棒 | 不捧场/说来听听 |
| 工作 | 工作/代码/bug/git | 退到背景不打扰 |

每个场景注入 ~100 字，按需触发。

## 架构变更

build_messages() 的 system prompt 结构从三层改为四层：
```
Layer 1: 静态人设（cache_control: ephemeral）
Layer 2: 今日上下文（cache_control: ephemeral，60s TTL）
Layer 3: 场景上下文（动态，无缓存）
Layer 4: 记忆浮现（动态，无缓存）
```

基准 system prompt：~3000 字（场景/记忆按需追加 ~100-500 字）
